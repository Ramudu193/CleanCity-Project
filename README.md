# 🌆 Clean City – Community Driven Complaint & Cleanup Platform

Clean City is a full-stack web application that allows citizens to raise complaints about city issues and enables volunteers to resolve them by uploading proof. The system includes admin verification, notifications, and a reward/engagement mechanism.

---

## 🚀 Tech Stack

### Frontend
- React (Vite)
- JavaScript
- CSS

### Backend
- Java 17
- Spring Boot
- Spring Data JPA
- REST APIs

### Database
- MySQL

---

## 📌 Features

- 🔐 User Registration & Login (JWT Authentication)
- 📝 Raise Complaints (with details & location)
- 👨‍🔧 Volunteer System to solve complaints
- 🎥 Upload Cleanup Proof (Video/Image)
- ✅ Admin Approval / Rejection of proofs
- 🔔 Notification System (on approval/rejection)
- 📊 Leaderboard (based on completed tasks)
- 📈 Complaint Status Tracking (PENDING → RESOLVED)
